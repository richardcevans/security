# Introduction

## About this Workshop

### Overview

Estimated Time: 60 minutes

This workshop environment is dedicated to Oracle Database Security features and
Oracle Deep Data Security.

In this lab, you use a local HR web application to connect directly to Oracle
Database as database end users. Oracle Database activates each user's Deep Data
Security data roles at login, and data grants enforce what rows and columns the
user can see or update.

### Components

During this workshop, you use these resources:

- SSH Terminal Client
- Oracle SQL*Plus
- End User Web HR App
- Oracle AI Database 26ai Free

Complete the setup steps in the lab before opening the web application. The app
expects the baseline Deep Data Security objects, end users, data roles, and data
grants to exist in the target PDB.

### Objectives

In this workshop, you will:

- Prepare the local web application runtime.
- Connect as Oracle Database end users.
- Compare employee and manager access in the HR application.
- Review how Deep Data Security data grants enforce row and column access.
- Check diagnostics that show the database end user and active data roles.

You may now [proceed to the next lab](#next).
